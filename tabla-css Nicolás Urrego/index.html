CSS


¿Que es?


CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico que se utiliza para definir la apariencia de los sitios web. Permite controlar el color, la fuente, la posición, las animaciones y los efectos visuales de los elementos de una página web.


¿Por que es importante?


Son una potente herramienta de desarrollo web que ayuda a separar la estructura de un sitio web de su presentación . La función principal de CSS es definir el estilo y el diseño de las páginas web, haciéndolas visualmente más atractivas, intuitivas y accesibles.



JAVA SCRIPT


¿Que es?


JavaScript (JS) es un lenguaje de programación de alto nivel, interpretado y orientado a objetos, ampliamente utilizado para crear páginas web interactivas y dinámicas, así como aplicaciones web y de escritorio.


¿Por que es importante?


JavaScript es un lenguaje de programación fundamental para el desarrollo web, ya que permite crear páginas interactivas. Es muy demandado en el mercado laboral.
